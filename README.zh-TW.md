# Tensei — キャラクターが転生してきた件

**[日本語](README.ja.md) | 繁體中文 | [简体中文](README.zh-CN.md) | [English](README.md)**

一款讓你真正能和日本網路小說角色對話的 Chrome 擴充功能。

每個角色只知道你設定的章節為止發生的事。不會有劇透，有真實記憶，個性也完全是那個角色的樣子。

---

## 適合這樣的人

- 想和喜歡的角色好好聊聊
- 不希望角色知道後面章節的發展
- 在意角色的口吻和個性是否忠實原著
- 不想把閱讀資料上傳到雲端

---

## 主要功能

**對話**
- 角色只記得你指定章節以前的事（防止劇透）
- 長時間的對話會自動壓縮摘要，不會失去重要脈絡
- 每次回覆都會自動引用原文中相關的段落

**角色設定**
- 可細緻設定人格、說話方式、口頭禪、喜好與禁忌
- 依故事時期做版本管理（少年時期、成長後等）
- 登錄聲音樣本與台詞範例，讓回應更有一致性

**演出模式**
- 「編劇模式」「導演模式」——讓 AI 生成場景劇本
- 「演員模式」——你自己也作為角色之一參與其中
- 「BTS 模式」——拍攝空檔，角色們私下閒聊

**隱私**
- 所有資料只存放在你的瀏覽器（IndexedDB）中
- 不會傳送到外部伺服器（除了你對 LLM API 的直接請求）

**多語言**
- 介面語言與 LLM 回覆語言可獨立設定（日文、繁體中文、簡體中文、英文）

## 安裝方式（開發版）

> 正在準備上架 Chrome Web Store。目前需要手動安裝。

```bash
git clone https://github.com/Chakotay-Lee/tensei-extension.git
cd tensei-extension/extension
npm install
npm run build
```

開啟 Chrome 的 `chrome://extensions`，啟用「開發人員模式」，點選「載入未封裝項目」，選擇 `extension/dist` 資料夾。

---

## 使用流程

1. **設定 LLM** — 前往 Settings，加入任意 OpenAI 相容的端點（OpenAI、OpenRouter、Ollama 等）與 API 金鑰
2. **匯入小說** — 開啟小說頁面，透過側邊欄匯入
3. **選擇角色開始對話** — 設定角色知道幾章以前的內容，然後開始

## 常見問題

**需要 API 金鑰嗎？**
需要。請自備 OpenAI、OpenRouter、Ollama 或其他 OpenAI 相容端點的 API 金鑰。這個 App 不提供 LLM 服務。

**我的資料存在哪裡？**
全部存在你的瀏覽器（IndexedDB）中，不會傳送到外部伺服器（除了你對 LLM 的 API 請求）。

**支援哪些小說平台？**
匯入功能針對 Kakuyomu 和小説家になろう（Syosetu）最佳化，也可以直接貼上文字使用。

**有些角色設定欄位鎖起來無法修改？**
那是該作品的作者提供了官方角色資料，並設定為禁止修改的欄位。這是為了尊重作者意圖的機制。

**我是作者，想提供官方角色設定**
請參閱[作者指南](docs/author-guide.zh-TW.md)。透過 Portal 完成認證後，就可以登錄官方角色設定與章節摘要。

**找不到想看的作品？**
向作者建議加入 Tensei，或者自行貼上文字匯入。
