# Tensei — キャラクターが転生してきた件

**[日本語](README.ja.md) | [繁體中文](README.zh-TW.md) | 简体中文 | [English](README.md)**

一款让你真正能和日本网络小说角色对话的 Chrome 扩展。

每个角色只知道你设定章节以前发生的事。没有剧透，有真实记忆，性格也完全是那个角色的样子。

---

## 适合这样的人

- 想和喜欢的角色好好聊聊
- 不希望角色知道后续章节的发展
- 在意角色的语气和性格是否忠实原著
- 不想把阅读数据上传到云端

---

## 主要功能

**对话**
- 角色只记得你指定章节以前的事（防止剧透）
- 长时间对话会自动压缩摘要，不会丢失重要上下文
- 每次回复都会自动引用原文中相关段落

**角色设定**
- 可细致设定人格、说话方式、口头禅、喜好与禁忌
- 依故事时期做版本管理（少年时期、成长后等）
- 注册声音样本与台词示例，让回应更有一致性

**演出模式**
- "编剧模式""导演模式"——让 AI 生成场景剧本
- "演员模式"——你自己也作为角色之一参与其中
- "BTS 模式"——拍摄间隙，角色们私下闲聊

**隐私**
- 所有数据只存放在你的浏览器（IndexedDB）中
- 不会传送到外部服务器（除了你对 LLM API 的直接请求）

**多语言**
- 界面语言与 LLM 回复语言可独立设定（日语、繁体中文、简体中文、英语）

## 安装方式（开发版）

> 正在准备上架 Chrome Web Store。目前需要手动安装。

```bash
git clone https://github.com/Chakotay-Lee/tensei-extension.git
cd tensei-extension/extension
npm install
npm run build
```

打开 Chrome 的 `chrome://extensions`，启用「开发者模式」，点击「加载已解压的扩展程序」，选择 `extension/dist` 文件夹。

## 使用流程

1. **设置 LLM** — 前往 Settings，添加任意 OpenAI 兼容的端点（OpenAI、OpenRouter、Ollama 等）与 API 密钥
2. **导入小说** — 打开小说页面，通过侧边栏导入
3. **选择角色开始对话** — 设定角色知道几章以前的内容，然后开始

## 常见问题

**需要 API 密钥吗？**
需要。请自备 OpenAI、OpenRouter、Ollama 或其他 OpenAI 兼容端点的 API 密钥。这个 App 不提供 LLM 服务。

**我的数据存在哪里？**
全部存在你的浏览器（IndexedDB）中，不会传送到外部服务器（除了你对 LLM 的 API 请求）。

**支持哪些小说平台？**
导入功能针对 Kakuyomu 和小说家になろう（Syosetu）优化，也可以直接粘贴文字使用。

**有些角色设定栏位被锁定无法修改？**
那是该作品的作者提供了官方角色数据，并设定为禁止修改的字段。这是为了尊重作者意图的机制。

**我是作者，想提供官方角色设定**
请参阅[作者指南](docs/author-guide.zh-CN.md)。通过 Portal 完成认证后，就可以登录官方角色设定与章节摘要。

**找不到想看的作品？**
向作者建议加入 Tensei，或者自行粘贴文字导入。
