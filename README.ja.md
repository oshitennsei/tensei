# キャラクターが転生してきた件

好きな小説のキャラクターと、実際に会話できる Chrome 拡張機能です。

キャラクターは「あなたが指定した章まで」の出来事しか知りません。ネタバレなし、記憶あり、性格もちゃんとそのキャラのまま。

---

## こんな人に向いています

- 好きなキャラと語り合いたい
- 読んでいる章のキャラに、物語の続きを知られたくない
- キャラの口調や価値観を大切にしたい
- データをクラウドに送りたくない

---

## 主な機能

**会話**
- キャラクターは指定した章までの記憶しか持たない（ネタバレ防止）
- 長い会話は自動で圧縮・要約され、文脈が失われない
- 関連する原文パッセージが自動で参照される（RAG）

**キャラクター設定**
- 人格・話し方・口グセ・好きなこと・嫌いなことを細かく設定
- キャラの「時代」ごとにバージョン管理（少年時代 / 成長後 など）
- 声サンプル・セリフ例を登録して一貫性を高める

**演出モード（演じる・書く・監督する）**
- LLM にシーンを生成させる「脚本家モード」「監督モード」
- あなた自身がキャラの一人として参加する「キャスト（出演）モード」
- 撮影の合間にキャラたちが雑談する「BTS モード」

**プライバシー**
- すべてのデータはあなたのブラウザ内 IndexedDB にのみ保存
- サーバーへの送信はなし（LLM API への直接リクエストを除く）

**多言語対応**
- UI と LLM の応答言語を独立して設定可能（日本語・繁体中国語・簡体中国語・英語）

## インストール方法（開発版）

> Chrome Web Store への公開準備中です。現時点では手動インストールが必要です。

```bash
git clone https://github.com/Chakotay-Lee/tensei-extension.git
cd tensei-extension/extension
npm install
npm run build
```

Chrome で `chrome://extensions` を開き、「デベロッパーモード」をオンにして、`extension/dist` フォルダを「パッケージ化されていない拡張機能を読み込む」から追加してください。

---

## 使い方の流れ

1. **LLM の設定** — Settings で OpenAI・OpenRouter・Ollama など任意の API エンドポイントと API キーを登録します
2. **作品を追加** — 小説のページを開いてサイドパネルから取り込む
3. **キャラを選んで話す** — どの章まで知っているかを設定して会話開始

## よくある質問

**Q. API キーが必要ですか？**
はい。OpenAI や OpenRouter などの API キーをご自身で用意してください。このアプリは LLM を提供しません。

**Q. データはどこに保存されますか？**
すべてあなたのブラウザ内（IndexedDB）に保存されます。外部サーバーには送信されません（LLM への API リクエストを除く）。

**Q. 特定の作品に対応していますか？**
取り込み機能は Kakuyomu・小説家になろうに最適化されていますが、テキストを直接貼り付ける方法でも使えます。

**Q. キャラの設定の一部が変更できません**
その作品の作者が「公式設定」として提供し、変更禁止にしているフィールドです。作者の意図を尊重するための仕組みです。

**Q. 作者ですが、公式のキャラ設定を提供したいです**
[作者向けガイド](docs/author-guide.ja.md) をご覧ください。ポータルでの認証後、キャラ設定・章サマリーを公式データとして登録できます。

**Q. 好きな作品がありません。どうすれば？**
その作品の作者に「Tensei への登録」をリクエストするか、テキストを貼り付けてご自身で取り込んでください。
