## Tensei に参加するとどうなるか

**何もしなくても**、読者はあなたの作品を自分でインポートして、自作のキャラ設定でこのアプリを使えます。

**あなたが作者として登録する**と、次のことができます：

- **公式キャラ設定の提供** — 人格・話し方・禁止ワードなどを「公式データ」として配信できる
- **フィールドのロック** — 読者に変更してほしくない設定（口調・人格など）を変更不可にできる
- **章サマリーの提供** — 各章の要約文を公式提供することで、より正確な文脈理解が可能になる
- **公式バッジ** — 読者の画面に「作者提供」と表示される

**ロックされた設定は何ですか？**
「人格（persona）」「話し方（speech_style）」「やらないこと（will_not_do）」「禁止トピック（forbidden_topics）」の 4 種類。読者はロックされたフィールドを上書きできず、閲覧のみ可能になります。ロックしていないフィールドは読者が自由に補足・拡張できます。

## なぜ認証が必要なのか

誰でも「この作品の作者です」と名乗れてしまうと、偽の公式設定が出回るリスクがあります。Tensei は以下の 3 段階で本人確認を行います：

1. **メールアドレスの確認** — マジックリンクでメールアドレスを所有していることを確認
2. **プラットフォームアカウントの確認** — Kakuyomu の活動報告または小説家になろうの活動報告に確認コードを掲載することで、そのアカウントの所有者であることを確認
3. **管理者による審査** — 運営が内容を確認して承認

自動化できる部分も多いですが、ToS 問題（スクレイピング）を避けるため、最終承認は人間が行っています。

## 登録手順

### ステップ 1 — ポータルにアクセス

[https://tensei-portal.pages.dev](https://tensei-portal.pages.dev) を開き、メールアドレスを入力して「マジックリンクを送信」をクリックします。

### ステップ 2 — メールを確認

届いたリンクをクリックすると、ダッシュボードに移動します。

### ステップ 3 — 作品情報を登録

ダッシュボードで以下を入力します：
- 作品タイトル（小説サイト上と同一のもの）
- プラットフォーム（Kakuyomu / なろう / その他）
- 作品ページの URL

### ステップ 4 — 本人確認コードを掲載

ダッシュボードに表示される確認コード（例：`TENSEI-XXXXXX`）を、**作品の活動報告・作者ノートなど公開されているページ**に記載してください。

確認できたら「確認する」ボタンを押してください。

### ステップ 5 — 審査を待つ

管理者が確認し、問題なければ承認メールが届きます。通常数日以内です。

## 承認後にできること

### キャラクター設定の配信

ダッシュボードの「キャラクター・サマリーを管理」からキャラを追加できます。

| フィールド | 説明 |
|---|---|
| 名前（name） | キャラクターの正式名称 |
| 人格（persona） | 性格・行動パターン・価値観・読者への接し方（400〜500 字程度） |
| 話し方（speech_style） | 口調・語尾・口グセ（50〜100 字） |
| やらないこと（will_not_do） | キャラがやらないこと（1 行 1 項目） |
| 禁止トピック（forbidden_topics） | 話してほしくない話題（1 行 1 項目） |
| ロックするフィールド | 上記のうち、読者に変更させたくないもの |

**ヒント：** `will_not_do` と `forbidden_topics` は常に「追加のみ可能」です。読者が独自のものを追加することはできますが、あなたが設定したものを削除することはできません。

### 章サマリーの配信

各章に公式サマリーを提供できます。読者の環境では、LLM が自動生成したサマリーよりも優先して表示・参照されます。

## よくある質問

**Q. 登録しないと使えませんか？**
いいえ。登録しなくても読者はアプリを使えます。登録は「公式設定を届けたい作者」向けの機能です。

**Q. 作品の文章そのものは配信されますか？**
いいえ。このシステムが配信するのはキャラクターの設定情報と章サマリーのみです。原文テキストは読者が自分でインポートします。

**Q. ロックしたフィールドを後から外せますか？**
はい。ダッシュボードでいつでも変更できます。

**Q. 同人・二次創作の作者ですが登録できますか？**
原作の権利を侵害しない範囲であれば登録可能ですが、一次創作の作者に優先して対応します。

**Q. 拒否されることはありますか？**
本人確認ができない場合や、内容が適切でないと判断した場合に拒否されることがあります。詳しくはお問い合わせください。

**Q. 一度提出したキャラ設定を削除できますか？**
はい。ダッシュボードから削除できます。削除後、読者の端末には次回の同期時に反映されます。
