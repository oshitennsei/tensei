Features

読書体験を、
会話体験へ。

ただのチャット bot ではありません。キャラクターが本当に「その世界の住人」として話してくれます。

📖
章単位の正確な記憶
「第13章まで」と指定すると、キャラクターはその章までの出来事しか知りません。ネタバレは一切なし。
🔍
RAG 自動参照
毎回の返答に、原文の関連パッセージが自動で参照されます。キャラクターは小説の細部を正確に覚えています。
🗜️
多段階の会話圧縮
長い会話でも重要な文脈は失われません。多段階の圧縮アルゴリズムが会話の流れを維持します。
🎭
演出モード
シーンを生成、物語を監督、キャラクターとして参加。創作の新しい楽しみ方を提供します。
🎬
BTS モード（後台）
撮影の合間、キャラクターたちが役を離れて素顔でグループ雑談。知らなかった一面が見えてきます。
⚙️
自前の LLM を使う
OpenAI・OpenRouter・Ollama・任意の OpenAI 互換エンドポイントに対応。APIキーはあなたが管理します。

How to use

3ステップで
すぐ始められる

01
拡張機能をインストール
Chrome ウェブストアから Tensei をインストール。
アカウント登録は不要です。
Chrome に追加 →
02
小説をインポート · LLM を設定
カクヨム・なろうのページを開いて章をインポート、またはテキストを貼り付け。
設定画面で使いたい LLM の API キーを入力します。
OpenAI / OpenRouter / Ollama 対応
03
キャラクターを選んで話す
キャラクターを選んで「何章まで知っているか」を設定するだけ。
あとはキャラクターが本当に話しかけてきます。

作者の方へ

カクヨム・なろうで作品を公開している作者は、Tensei ポータルに登録することで 公式キャラクター設定を配信できます。読者の拡張機能に自動的に反映され、 特定フィールドをロックして声を守ることもできます。

  1. メールアドレスを確認（マジックリンク）
  2. 作品情報を入力し、確認コードを「作者ノート」に投稿
  3. 管理者が確認・承認（通常数日以内）
  4. キャラクター設定を提出 → 読者の拡張機能に反映
