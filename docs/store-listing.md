## Detailed description

### English

Tensei lets you have real conversations with characters from Japanese web novels — as if they just stepped out of the story.

**How it works**
Pick a character and set a chapter cutoff. The character only knows events up to that point — no spoilers, even if you've read further. Long conversations are automatically compressed so context doesn't drop off. Relevant passages from the source text are retrieved and referenced with every response.

**Key features**
- Chapter-accurate memory — characters only know what you let them know
- Multi-tier conversation compression — nothing important gets lost in long chats
- RAG — relevant novel passages injected automatically per message
- Character versioning — talk to a character as their younger self, or after a major turning point
- Performance mode — generate scenes, direct the story, or play a character yourself
- BTS mode — watch characters chat casually between scenes
- Bring your own LLM — OpenAI, OpenRouter, Ollama, or any OpenAI-compatible endpoint
- All data stays in your browser — no servers, no accounts required

**Supported platforms**
Chapter import is optimized for Kakuyomu and Syosetu (小説家になろう). You can also paste text directly.

**Privacy**
Everything is stored in your browser's IndexedDB. No data is sent to external servers except the direct API calls to your configured LLM provider. No analytics, no accounts, no telemetry.

**For novel authors**
Authors can register through the Tensei portal to publish official character settings and lock certain fields to preserve their characters' voices. See the author guide for details.
